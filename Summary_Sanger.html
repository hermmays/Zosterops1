Sanger sequencing summary statistics and trees

Goals

What follows is our workflow for the general descriptive analysis of the Sanger sequencing data set for Zosterops. In this analysis we will describe the variation across individual Sanger sequenced loci and construct trees for each of these loci using distance-based and maximum likelihood methods.

Getting started

We are using R version 4.4.2, RStudio version 2024.09.1 for the analysis. Quarto version 1.6.39 for Mac OS was used to document the workflow. Set the working directory in RStudio to one where the necessary phased Sanger sequencing alignments and other data are accessible.

Data files

This dataset includes Sanger sequenced polymerase chain reaction (PCR) amplicon alignments for two mitochondrial DNA (mtDNA) and six protein coding nuclear loci (nuDNA). Prior to this analysis Sanger sequencing ab1 files were uploaded to GeneiousⓇ 8.1.9 for editing, alignment, and quality control. Multiple sequencing runs for each amplicon were assembled and assembled loci were aligned across individual samples using Geneious Alignment. Alignments for each locus were trimmed and edited in Geneious and the identity of each locus confirmed with a megablast search. The reading frame was set by eye by avoiding frames with stop codons and a blastp search was done on the translation consensus for each alignment to confirm the locus identity. Alignments were exported as fasta files.

Alignments of diploid cdDNA loci were phased using PHASE 2.1.1 (Stephens, Smith, and Donnelly 2001). First, convert the alignment fasta files from Geneious to the inp format for PHASE 2.1.1 using the online SeqPHASE tool (Flot 2009). At the ‘Genotypes to be phased’ prompt click the ‘Browse’ button and select the appropriate fasta alignment. Click the ‘Run SeqPHASE’ button. Once step 1 of the SeqPHASE process is completed retain the seqphase.const file in addition to the seqphase.inp file and rename each file appropriately to match the locus name (e.g. B3GNT9_seqphase.inp and B3GNT9_seqphase.const for locus B3GNT9).

Move the inp file from SeqPHASE to the directory with PHASE 2.1.1 and cd to that directory in the terminal. The inp file is used to run the following PHASE 2.1.1 command in the terminal. For each locus <numberseed> should be replaced with an arbitrary number selected by the user and LOCUSNAME replaced with the designated name for that particular locus. Because each locus is relatively short chose the S option at the modeling flag (-M) as it assumes no within-locus recombination. Additional parameters are explained in the documentation for PHASE 2.1.1 available here.

./PHASE -MS -d1 -S<numberseed> -p0.85 -q0.85 -x5 -X10 LOCUSNAME_seqphase.inp LOCUSNAME_seqphase.out 5000 2 500

Further analysis will require fasta alignments for each locus where each diploid individual is associated with two sequences corresponding to each of the two phased alleles determined in PHASE 2.1.1. Return to the online SeqPHASE tool and click the link to ‘step 2’. Select the appropriate out file generated by the PHASE 2.1.1 run along with the matching const file generated from step 1 in the SeqPHASE tool. The ‘Sort sequence’ box should be checked but the ‘Reduce sequences’ box should not be checked. Click the ‘Run SeqPHASE’ button and SeqPHASE will return a fasta file with the complete sequence alignment and two phased haplotypes designated by the project number for that individual sample followed by a and b respectively. Click the ‘Click here to download the generated Fasta file’ link at the bottom of the page and rename the file to match the locus name. Including the const file and unchecking the ‘Reduced sequences’ box will ensure the resulting fasta file includes the full length of the alignment, not just the variable sites.

Pairwise divergence

Determine the pairwise divergences among the sequences and display the 40 least divergent pairwise comparisons and the 40 greatest pairwise comparisons. Also focus on a particular pair of samples between Z. japonicus and Z. montanus (i.e. ZJal003a and ZMOvu001a) to get an idea of the mtDNA divergence within this focal group.

# Create a long format data frame
